# 🔐 Prompt Injection Guard

A lightweight security layer for detecting and mitigating prompt injection attacks in Large Language Model (LLM) applications.

## 🚀 Overview

As LLMs become integrated into real-world systems, they are increasingly vulnerable to **prompt injection attacks**—malicious inputs designed to override system instructions, extract sensitive data, or manipulate outputs.

This project applies **cybersecurity principles** to AI systems by detecting and filtering adversarial prompts before they reach the model.

## 🎯 Features

- 🔍 Detection of common prompt injection patterns  
- ⚠️ Risk scoring system (Low / Medium / High)  
- 🧠 Hybrid approach: rule-based + heuristic analysis  
- 🔒 Pre-processing layer for LLM pipelines  
- 📊 Simple output with explanation of detected risks
